Available Packages and Versions
Windows installer packages are marked win and are for use on Windows under Microsoft .Net 1.0 or 1.1. Zipped source packages (src) are provided for those who plan to contribute to the NUnit project or to modify NUnit for their own use own use. Beginning with NUnit 2.1.5, we provide a zipped package for use under Mono.

Production releases are stable releases that have gone through a beta review and had most of the problems fixed. Beta releases are subject to change as we receive feedback and are likely to have more bugs than the latest stable release. Most people will want to use a stable release for production projects unless they need some feature provided in a beta.
